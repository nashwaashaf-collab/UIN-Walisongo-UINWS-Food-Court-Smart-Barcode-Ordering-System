# UIN-Walisongo-UINWS-Food-Court-Smart-Barcode-Ordering-System
The UINWS Food Court Smart Ordering System modernizes dining at UIN Walisongo Semarang. By scanning table barcodes (QR codes), students and staff can browse multi-vendor menus, order, and pay instantly via QRIS/e-wallets. This contactless solution eliminates queues, speeds up service, and streamlines stall operations efficiently.

# How the System Works
[ Scan Table Barcode ] ➔ [ Browse Digital Menu ] ➔ [ Order & Pay Online ] ➔ [ Food Delivered / Ready ]
1. Scan the Barcode: Customers sit at any food court table and scan a unique QR code/barcode fixed to the tabletop using their smartphone camera.
2. Access Digital Menu: Scanning automatically opens a web-based menu tied specifically to that table number—no app installation required.
3. Customize & Place Order: Customers browse dishes across various campus vendors, customize their items, and add them to a single cart.
4. Seamless Payment: Payments are made directly through integrated digital channels (QRIS, E-wallets like GoPay/OVO/Dana, or mobile banking) or marked for cash payment.
5. Real-Time Kitchen Processing: Vendors receive orders immediately on their merchant dashboard, prep the food, and either notify the customer via SMS/WhatsApp or deliver it directly to the designated table.

# Key Features
Table-Specific Barcodes: Unique QR codes mapped to specific table numbers to ensure accurate delivery.
Multi-Vendor Browsing: Allows users to order from multiple food stalls within the same transaction.
App-Less Web Interface: Built as a Responsive Web App (PWA) accessible instantly via mobile browser.
E-Payment Integration: Native support for QRIS and popular Indonesian e-wallets.
Live Order Tracking: Status updates (e.g., Preparing, Ready for Pickup, Served).
Vendor Management Portal: Dedicated dashboard for food stall owners to manage inventory, prices, and incoming orders.
